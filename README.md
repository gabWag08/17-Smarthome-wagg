# Smarthome VR – Analoge Erstversion

## Überblick
Dieses Projekt ist eine erste **visuelle Grundstruktur** eines Smarthome-Konzepts in Virtual Reality.  
Die aktuelle Version stellt den **Rohbau** dar und dient als Basis für spätere Interaktionen und Smart-Home-Funktionen.

## Inhalt des Projekts
- VR-Template als Grundlage  
- **3 Räume** (Rohbau)  
- **3 Türen**  
- **2–3 Fenster**  
- Aufbau orientiert an der Skizze aus dem Unterricht

## Development / Target Platform
- **Engine:** Unity (VR Template)  
- **Entwicklungsplattform:** PC  
- **Zielplattform:** Meta Quest

## Playthrough
- Freies Bewegen durch die Räume (noch nicht implementiert)
- Besichtigung der räumlichen Aufteilung
